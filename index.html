Oracle-Based Student Housing Database

Tech Stack: Relational Database design | Oracle SQL | Relational Algebra | ER diagram | Oracle Cloud | Oracle Data Modeler | Excel | Oracle SQL Developer

Overview

This project involved designing and implementing a student housing database for a university's housing office. The goal was to efficiently store and manage data related to student housing, dormitory assignments, leases, staff responsibilities, and emergency contacts.

Project Details

  1. Database Design:
    • Defined the structure of the database to store essential student housing information, including:
      • Student details (name, major, home address, emergency contact, mentor)
      • Dorm and apartment assignments (current residence, lease details)
      • Housing facilities (dorm buildings, school apartments, staff responsibilities)
    • Used Oracle Data Modeler to create an ER diagram, mapping out entities, relationships, and constraints.
  2. Database Implementation:
    • Converted the ER diagram into SQL scripts to create tables in an Oracle Cloud database.
    • Defined primary keys, foreign keys, constraints, and relationships to ensure data integrity.
  3. Data Population:
    • Generated dummy student and housing data using a mix of manual input, AI-generated addresses, and Excel spreadsheets.
    • Imported the structured data into the database tables.
  4. Testing & Querying:
    • Used Oracle SQL Developer to write and execute various SQL queries to validate database functionality.
    • Designed queries to retrieve information such as:
      • Students currently living in dorms vs. apartments
      • Lease expiration dates and renewal needs
      • Staff assigned to specific buildings
      • Emergency contact details for students
